Detailed Description

un Axe permet d'avoir un combiné de touches assignées pour une même action résultant une valeur comprise entre -1 et 1

Parameters
namenom de l'axe
positiveButtoninput permettant d'obtenir la valeur 1
altPositiveButtoninput alternatif permettant d'obtenir la valeur 1
negativeButtoninput permettant d'obtenir la valeur -1
altNegativeButtoninput alternatif permettant d'obtenir la valeur -1
joystickIdindex du joystick d'un gamepad (si celui ci reste sur -1, aucun joystick ne sera assigné)
gamepadIdindex du gamepad. Si un joystick est définit mais que le gamepad reste à -1, alors l'axe vérifiera tous les gamepads (les 2 gamepads)
invertpermet d'inverser le résultat (-1 -> 1, 1 -> -1)
